CourtAPI Training Site Example Cases

The CourtAPI Training site uses the PACER test and training courts to fetch data from PACER. These courts do not contain the same data set as the live PACER sites, which can make locating cases to use for testing difficult. For example, some test courts do not contain any cases filed after 2007. Some courts only contain a few fake cases.

In order to make testing of CourtAPI client applications easier, we have provided a list of example court cases that can be found in the PACER test or training courts below.
